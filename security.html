LxRoot

Security Features

Designed for the modern host, LxRoot has all the tools and features you need to build and scale a cluster to meet your needs.

Let's Explore ↓

Change Tracker

Track your source code changes and restore them if needed.

App containerisation

Each stand alone application is containerised to protect other apps in your cluster from any vulnerabilities.

Brute force protection

Configure rate limits and block/allow lists for email addresses and IPs to help protect your cluster.

ModSecurity (OWASP)g

Each server automatically inherits your global settings - make changes per-server where needed.

WordPress toolkit

Enable or disable ModSecurity for each server, customise default OWASP settings, add custom rules using an update the running OWASP version.

SSL / TLS

All applications and services on LxRoot fully support SSL/TLS.
